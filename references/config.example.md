# Local configuration

Copy this file to `config.local.md` and replace every placeholder. Never commit `config.local.md`.

- Zotero source collection: `your-inbox-collection`
- Vault root: `/path/to/your/ObsidianVault`
- Literature folder: `10_Literature`

On a second computer, create a separate local configuration with that computer's Vault path. The collection name and literature folder can remain the same if you want a consistent workflow.
